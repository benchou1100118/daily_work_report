# 每日工作匯報 PySide2 UI

此專案提供 Python 3.8.10 / PySide2 桌面程式，讓人員自行註冊帳號、登入後依日期填寫每日工作匯報，並將 CSV 上傳到指定 FTP 交接資料夾。

## FTP 設定

程式預設連線資訊如下：

- FTP：`ftp://192.168.153.7`
- 帳號：`User`
- 密碼：`123456`
- 資料夾：`Largan_Machine_data/個人資夾/@交接資料/每日工作匯報`

## 使用方式

1. 建立 Python 3.8.10 環境。
2. 安裝套件：
   ```bash
   pip install -r requirements.txt
   ```
3. 啟動程式：
   ```bash
   python main.py
   ```
4. 未註冊者點選「註冊新人員」，輸入姓名、工號與密碼完成註冊。
5. 以工號與密碼登入。
6. 登入後可使用小行事曆點選報告日期；若該人員該日期已有本機備份，畫面會自動載入先前儲存/上傳的內容。
7. 填寫或修改工作內容後按「儲存並上傳FTP」。同一人員同一日期的報告會覆蓋成最新內容。
8. 登入後可使用「更改密碼」更新個人密碼。

## 人員註冊與帳號資料

人員不再由程式內或獨立名單預先建檔，改由使用者在介面中註冊。註冊完成後，帳號與密碼雜湊會保存在本機 `users.json`。

`users.json` 由程式自動建立與維護；請避免手動修改密碼雜湊欄位。

## 日期報告與本機備份

每次送出會先在 `reports/` 建立或覆蓋該人員該日期的 CSV 備份，再上傳至 FTP。檔名格式為：

```text
YYYYMMDD_工號_姓名.csv
```

若 FTP 連線失敗，程式會保留本機 CSV 並顯示錯誤訊息。再次登入並點選相同日期時，會自動載入本機備份內容，方便修改後重新上傳。
